Getting Started

Step 1

Register

Start by registering a developer account. Registering provides you access to all developer resources.

Step 2

Development Environment

Set up your development environment. Download plug-ins for popular IDE's that assist with code completion and debugging.

Step 3

Read

Review documentation and videos and screencasts, and keep the API reference at hand.

Step 4

Build

Take your ideas and run with them, being sure to follow best practices. Can't find the answer to a question in the API reference or FAQ? Look to the developer support forums.

Step 5

Submit

Once you're done, send in your application for review. Wait a bit while our editors review your code.

Step 6

Market

Make sure interested users can find your application by augmenting it with categorical and domain information.

Step 7

Support

As your application gains users, provide support for their issues and questions and feature requests.
