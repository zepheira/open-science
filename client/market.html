Search Applications
This is the Open Science Application Marketplace where a user can find and purchase applications that make scientific research faster and easier. Applications may be used to enhance an entire search result set or augment content of one article. Use the Marketplace to browse for applications by research domain, application type, creator and other properties. Special indications are made for Open Science "Editor's Pick" or "Featured Application" to draw user attention.
Each application is rated by Marketplace users, indicating those considered most useful.
To further give a sense of an application's value, a breakdown is shown of how many times the application has been purchased. This breakdown is segmented by user type.
To see an application in context as it is applied to a set of results or an individual result, launch a preview with the preview button. Note that the button is not active in this mockup.
Applications have a price, though some may be free. To add one to your applications, first click the Add button, which will pop up the option to either purchase it yourself or notify your librarian that you'd like to submit the application for institutional requisition. Either way, it will then be placed in your application cart.
Added applications are placed in the application cart. They can be removed here or from their place in the list; having any applications present reveals a purchasing link.
Further tools to restrict the set of applications displayed are provided to the right in the set of facets, including by rating, price, editor's pick, creator, and others.
The tutorial continues by clicking on any of the applications. They will all lead to the same place in this mockup system.
